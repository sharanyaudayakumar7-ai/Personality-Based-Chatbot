
# 🧠 Personality Chatbot 😎✨

A fun and interactive web-based chatbot that responds with different personalities like Savage 😈, Sweet 🥺, Sarcastic 🙄, and Motivational 🚀.

---

## 💛 Features

* 🎭 Multiple personality modes
* 😂 Random funny & engaging responses
* 🎨 Aesthetic yellow-themed UI
* 💬 Chat bubble design
* 🔊 Button sound effects
* ✨ Smooth animations

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/personality-chatbot.git
```

2. Navigate to the project folder:

```
cd personality-chatbot
```

3. Install dependencies:

```
pip install flask
```

4. Run the app:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
